# Configuration (config/)

This directory contains configuration files and environment settings.

## Purpose
- Environment configuration files
- Model hyperparameters
- Application settings
- Deployment configurations

## Structure
Store all configuration files, including:
- `.env` files (do not commit sensitive data)
- YAML/JSON configuration files
- Constants and settings modules

## Note
Remember to add sensitive configuration files to `.gitignore` to prevent committing credentials or secrets.
