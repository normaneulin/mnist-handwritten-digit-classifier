# DL-Classifier-Web-Application
Trained Deep Learning Classifier as a Web Application

## Project Structure

This repository is organized to support reproducibility, scalability, and collaborative development workflows:

```
DL-Classifier-Web-Application/
├── src/                # Model integration and core logic
├── app/                # Streamlit web application interface
├── utils/              # Preprocessing and utility functions
├── docs/               # Documentation, reports, and screenshots
├── tests/              # Validation tests and test fixtures
└── config/             # Configuration files and environment settings
```

### Directory Descriptions

- **src/**: Contains the core model integration code, including model architecture, training scripts, and inference logic.
- **app/**: Houses the Streamlit application for the web interface, including UI components and user interaction handlers.
- **utils/**: Provides utility functions for data preprocessing, transformations, and common helper functions.
- **docs/**: Stores project documentation, reports, screenshots, and user guides.
- **tests/**: Contains unit tests, integration tests, and validation scripts to ensure code quality.
- **config/**: Holds configuration files, environment settings, and hyperparameters.
